# 《废土花开》开发任务清单 - 阶段三 (Month 3: 封闭测试 Alpha - 净化战)

- [ ] **任务 21：“毒物农场”高风险高收益策略机制**
  - 允许植物在辐射区或防卫失败时变异为“剧毒变异品种”。
  - 实现剧毒品种产出稀有材料，用于升级终极武器或背包。

- [ ] **任务 22：后悔药机制：紧急净化炸弹**
  - 实现紧急净化炸弹的制作逻辑，强制要求消耗极其稀缺的“高能纯水”（牺牲通关进度）。
  - 实现投掷与爆炸逻辑，清空目标区域内所有怪物与毒物农场，保住庇护所安全。

- [ ] **任务 23：净化节点与源生净化树生成**
  - 在地图固定位置生成重度污染节点。
  - 实现源生净化树种子的稀有获取途径及专属种植限制。

- [ ] **任务 24：动态 7 天净化防卫战 (前两天)**
  - 第 1-2 天：纯净水浇灌，仅有轻度骚扰，风平浪静。
  - 实现日常纯净水收集与浇灌交互逻辑。

- [ ] **任务 25：第 3 天局部辐射风暴与第一波虫潮**
  - 触发小高潮事件：改变当天光照颜色，生成局部辐射风暴。
  - 刷出针对净化树的第一波虫潮，测试玩家防守压力。

- [ ] **任务 26：第 4-6 天防守与高能纯水机制**
  - 允许玩家在探索中获得极度稀有的“高能纯水”。
  - 使用高能纯水直接跳过 1 天的净化树生长期。

- [ ] **任务 27：第 7 天 Boss 战机制设计**
  - 触发净化大高潮防卫战。
  - 设计并生成 Boss 怪物，考验玩家前期的武器与陷阱储备。

- [ ] **任务 28：生态净化事件视觉演出 (The Purge Moment)**
  - 防守成功点击净化树，爆发绚丽冲击波特效。
  - 实现 TileMap 的动态翻转（紫黑色腐败泥土 -> 翠绿色草地）：建议使用 `TileMapLayer.set_cell()` 结合 `Tween` 实现 0.8 秒的渐变动画效果，极大提升视觉冲击力，并动态清除战争迷雾。

- [ ] **任务 29：光照与粒子视听转换**
  - 音乐切换为悠扬空灵的吟唱。
  - 空气中的 GPUParticles2D 浑浊粒子消散，全局光照 (CanvasModulate) 从阴冷变为温暖橙黄。

- [ ] **任务 30：辐射惩罚与抗辐射药剂系统**
  - 根据玩家所在污染区域浓度动态增加辐射值，超标触发惩罚（MVP 阶段仅做简单数值叠加与扣减，EA 阶段再加入屏幕边缘绿噪等视觉 Debuff）。
  - 实现消耗基础材料酿造抗辐射药剂，提供短暂探索免疫 Buff。

# 《废土花开》开发任务清单 - 阶段一 (Week 1: 玩具箱 Demo)

- [x] **任务 1：搭建项目基础框架**
  - 设定游戏分辨率、项目基本设置。
  - 准备 3 个主角模型预设（猫科、熊科、兔/鼠科，无复杂捏脸），配置基础 8 向移动。

- [x] **任务 2：实现全局时间系统 (简单昼夜与天数)**
  - 编写全局单例脚本管理游戏时间。
  - 实现简单的日历（天数统计）和昼夜交替流逝逻辑。

- [ ] **任务 3：扩展玩家基础属性与 UI**
  - 增加体力值 (Stamina)、生命值 (HP) 和辐射值 (Radiation) 属性。
  - 搭建主界面 HUD（血条、体力条、时间显示）。

- [ ] **任务 4：Furry 元素轻量化表现**
  - 为不同预设角色增加特定的动画细节（如猫科待机舔爪子、睡觉蜷缩；熊科睡觉呼噜声与 UI 气泡）。

- [ ] **任务 5：坚决手搓 MVP 关卡地图 (Hand-crafted Map)**
  - 拒绝程序化生成，建立多层 TileMap 并在 Godot 中手绘固定大小的微型地图。
  - 左侧为庇护所农田，右侧为通往固定“源生净化树”节点的小路，固定所有刷怪点与资源位置。
  - 预留 3-4 个“未来扩展地块”（使用不同的 Terrain Layer 标记），以便后续无缝接入净化翻转演出。

- [ ] **任务 6：建立交互框架**
  - 为玩家添加交互检测区域 (Area2D / RayCast2D)。
  - 实现对周围环境、工作台、土地的检测与交互判定。

- [ ] **任务 7：基础背包与快捷栏系统**
  - 编写背包数据结构。
  - 在 UI 底部实现快捷栏，支持手持工具（锄头、水壶）、武器和种子的切换。

- [ ] **任务 8：基础种田系统 (挖地与浇水)**
  - 将玩家动作与 TileMap 交互连接，实现挥锄头挖地（转化为“可耕种土地”）。
  - 实现水壶浇水机制，改变土地状态并记录湿润度。

- [ ] **任务 9：定义灵魂数据结构 (PlantResource) 与状态机**
  - 使用 Godot 的 Resource 类定义 `PlantResource`，管理植物各阶段的基础数据。
  - 创建通用的植物实体，包含状态：种子 -> 幼苗 -> 成熟 -> 枯萎/收获。
  - 接入全局时间系统，根据天数推进植物生长进度。

- [ ] **任务 10：完成玩具箱 Demo 闭环**
  - 实现核心植物“血浆果”的种植、生长、收获流程。
  - 实现食用血浆果恢复生命和体力的效果，验证基础种田循环。

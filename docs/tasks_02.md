# 《废土花开》开发任务清单 - 阶段二 (Month 1: 垂直切片与战斗闭环)

- [ ] **任务 11：战斗系统基础 (Hitbox & Hurtbox)**
  - 搭建标准化的攻击判定框和受击判定框组件。
  - 实现玩家和敌人的受击判定、扣血与击退效果。

- [ ] **任务 12：核心武器实现：藤蔓鞭**
  - 实现种植与收获“藤蔓鞭”，可装备在快捷栏。
  - 实现藤蔓鞭的近战多段伤害逻辑与挥砍动画。

- [ ] **任务 13：极致打击感与交互反馈优化 (Juice)**
  - 拔草仪式感：拔出植物时加入屏幕微震与“啵”声音效，植物化作流光飞入手中。
  - 战斗卡肉感 (Hit Stop)：武器击中怪物时，引擎时间瞬间降至 0.1（持续 0.05 秒），伴随白光闪烁。

- [ ] **任务 14：种田式武器修复与强化机制**
  - 修改武器耐久耗尽逻辑：不直接销毁，而是退化为“枯萎的握把（种子残骸）”存入背包。
  - 实现“滋养休眠舱”或“荧光修复土”特殊地块，允许将残骸种回。
  - 使用“变异腺体”施肥，次日恢复满耐久并带有毒属性强化。

- [ ] **任务 15：核心陷阱实现：荆棘地雷**
  - 实现“荆棘地雷”的种植。
  - 编写触发逻辑与伤害区域，用于防御夜晚边缘怪物。

- [ ] **任务 16：基础敌人框架与冲锋型怪物**
  - 搭建敌人状态机 AI（巡逻、追逐、攻击）。
  - 实现“变异野猪”：直线高伤害冲锋，撞墙产生晕眩硬直。

- [ ] **任务 17：投射型怪物实现**
  - 实现“吐酸液变异花”：远程攻击预判玩家走位。

- [ ] **任务 18：夜间刷怪与昼夜难度切换**
  - 实现昼夜差异：白天安全，夜晚废土怪物狂暴、辐射潮汐上升。
  - 编写刷怪管理器（建议实现为 `MonsterSpawner` 单例），控制夜间边缘怪物的生成，并方便后续阶段直接复用给虫潮事件。

- [ ] **任务 19：战利品掉落与特殊种子获取**
  - 为敌人添加掉落物（变异腺体、特殊异星种子）。
  - 实现特殊种子种植出高级防守植物或武器材料的逻辑。

- [ ] **任务 20：完成垂直切片闭环测试 (前置依赖：任务 11-19)**
  - 跑通完整闭环：种出武器 -> 夜晚边缘打怪 -> 掉落特殊种子 -> 种下特殊种子。
  - 修复战斗与种植系统的交互冲突。

# 《废土花开》开发任务清单 - 阶段六 (Narrative, Modes & Endings)

- [ ] **任务 51：多游玩模式选择 (Game Modes)**
  - 在新建存档界面增加【田园模式】与【废土生存】（标准）选项。
  - 根据选择模式，全局调整怪物刷新逻辑（是否主动攻城）及辐射伤害计算公式（致命扣血 vs 头晕减速）。

- [ ] **任务 52：多结局判定系统 (Multiple Endings)**
  - 设置全局变量追踪玩家的“土地净化完成度”与“基因变异程度”。
  - 编写游戏结束触发条件，并根据数值分发坏结局（沉沦）、普通结局（绿洲）与真结局（初醒）的结算画面。

- [ ] **任务 53：碎片化叙事系统 (Fragmented Storytelling)**
  - 在特定废土地图预设交互式旧时代遗物（如废弃医院的八音盒、日记本）。
  - 开发文本阅读 UI 界面，展现信件、日记和背景故事。

- [ ] **任务 54：收藏图鉴 UI 框架 (Collection Book)**
  - 在主菜单或暂停菜单中添加图鉴入口。
  - 设计分栏界面（植物图鉴、怪物图鉴、旧时代遗物、信件与日记），展示已解锁与未解锁项。

- [ ] **任务 55：图鉴收集与属性增益关联逻辑**
  - 编写图鉴管理数据类，记录每一项的解锁状态。
  - 实现收集进度奖励机制（例如：解锁所有日记赋予玩家 +5% 抗辐射能力加成）。

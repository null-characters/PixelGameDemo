# 《废土花开》开发任务清单 - 阶段一 (Core Systems & Player)

- [ ] **任务 1：搭建项目基础框架**
  - 设定游戏分辨率、项目基本设置。
  - 规划目录结构（Scenes, Scripts, Assets, Resources）。
  - 配置全局输入映射（Input Map，包括移动、交互、攻击、切换物品等）。

- [ ] **任务 2：实现全局时间系统 (Time Manager)**
  - 编写全局单例脚本管理游戏时间。
  - 实现日历（天数统计）和昼夜交替流逝逻辑。
  - 暴露时间变更的信号，供其他系统（光照、怪物、植物生长）订阅。

- [ ] **任务 3：扩展玩家基础系统**
  - 在现有小白兔 8 向移动基础上，增加体力值 (Stamina)、生命值 (HP) 和辐射值 (Radiation) 属性。
  - 实现移动时消耗体力的基础逻辑。

- [ ] **任务 4：建立交互框架**
  - 为玩家添加交互检测区域 (Area2D / RayCast2D)。
  - 实现对周围物体（植物、敌人、工作台、土地）的检测与交互判定。

- [ ] **任务 5：实现基因特质系统**
  - 建立特质数据结构。
  - 实现开局选择：猫科（移速+15%，夜晚视野广）、熊科（伤害+20%，能搬运重物）、兔科/鼠科（种植速度+10%，易挖出稀有物品）。
  - 将特质属性加成应用到玩家对应的数值上。

- [ ] **任务 6：构建 UI 系统基础**
  - 搭建主界面 HUD。
  - 制作玩家状态栏（血条、体力条、辐射积累条）。
  - 制作当前时间、天数和昼夜状态的 UI 显示。

- [ ] **任务 7：实现基础背包与快捷栏系统**
  - 编写背包数据结构（Inventory Resource）。
  - 在 UI 底部实现快捷栏，用于存放工具（锄头、水壶）、种子和武器，支持按键切换当前手持物品。

- [ ] **任务 8：地形与 TileMap 重构**
  - 建立多层 TileMap 或 Terrain Sets。
  - 明确划分区域地块：“健康土地”（安全区）、“可耕种土地”、“不同浓度的辐射污染土地”（轻度、重度、核心）。

- [ ] **任务 9：昼夜视觉表现**
  - 添加 CanvasModulate 节点控制全局光照。
  - 配合时间系统，实现从清晨、正午、黄昏到深夜的环境颜色和平滑过渡。
  - 为玩家角色添加基础点光源（夜间视野）。

- [ ] **任务 10：环境氛围特效**
  - 为辐射污染区添加粒子特效（如飘浮的荧光绿辐射尘埃）。
  - 实现简单的“战争迷雾”视觉效果，遮挡未净化的深处区域。

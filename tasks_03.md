# 《废土花开》开发任务清单 - 阶段三 (The Purge & Settlement)

- [ ] **任务 21：抗辐射药剂与消耗品系统**
  - 建立简单的制作/酿造工作台交互逻辑。
  - 实现消耗材料制作抗辐射药剂，使用后赋予玩家一定时间的辐射免疫或抗性 Buff。

- [ ] **任务 22：拾荒与战利品系统 (Scavenging)**
  - 在废土区域随机生成可搜刮的旧时代遗物（废铁、垃圾堆）。
  - 为敌人添加掉落物列表（高科技零件、未知的异星种子）。

- [ ] **任务 23：净化节点机制 - 寻找与种植**
  - 在地图的重度污染区设定特定的“污染节点”。
  - 实现极其稀有的“源生净化树/塔”种子的获取条件及特定地点的种植限制。

- [ ] **任务 24：净化树的 7 天生长循环**
  - 将源生净化树的生长周期严格绑定游戏内的 7 天时间线。
  - 实现日常维护需求（如每天必须浇灌特定的“高能纯水”，否则停止生长或倒退）。

- [ ] **任务 25：净化节点防卫战**
  - 净化树生长期间，吸引周围变异生物仇恨。
  - 实现植物生命值系统，当被怪物啃食破坏后需重新开始净化流程。

- [ ] **任务 26：生态净化事件表现 (The Purge Event)**
  - 实现第 7 天净化树成熟的史诗级正反馈演出（特效发光、震屏）。
  - 动态替换周围的 TileMap，将污染地块转化为健康草地，清除战争迷雾，扩大安全区。

- [ ] **任务 27：庇护所基建升级系统**
  - 利用搜刮来的高科技零件，实现水泵（提供高能纯水）、发电机等设施的修复与升级。
  - 升级后解锁新的制作配方或提升庇护所的整体功能。

- [ ] **任务 28：NPC 搜救与招募系统**
  - 在废土特定区域放置流浪的兽化人 NPC。
  - 实现对话交互系统及完成特定条件（给食物、帮忙击败周围怪物）后的招募入队逻辑。

- [ ] **任务 29：NPC 聚落管理与自动化 (Auto-farming)**
  - 招募的 NPC 回到庇护所后，为其分配基础 AI 行为树。
  - 实现 NPC 自动执行浇水、收获已成熟基础作物的功能。

- [ ] **任务 30：NPC 守卫与协同防御**
  - 赋予部分战斗型 NPC 巡逻和攻击逻辑。
  - 当夜晚有外星怪物靠近庇护所边缘时，NPC 会自动协助防御。

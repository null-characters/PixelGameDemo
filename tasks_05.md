# 《废土花开》开发任务清单 - 阶段五 (Advanced Systems & Economy)

- [ ] **任务 41：进阶敌人 AI 行为 (Advanced AI)**
  - 实现变异野猪的直线冲锋攻击与撞墙晕眩逻辑。
  - 实现变异花的远程酸液投射与玩家走位预判。
  - 实现钻地沙虫的潜伏状态与触发地块判定。

- [ ] **任务 42：基因序列库与天赋树系统 (Gene Tree)**
  - 建立天赋树 UI 面板，分为猫科、熊科、兔/鼠科分支。
  - 实现消耗“异星变异果”或经验值解锁天赋（如强化夜视、重击）。
  - 将解锁的天赋属性永久应用到玩家角色上。

- [ ] **任务 43：种田式武器修复机制 (Weapon Repair)**
  - 增加“滋养休眠舱”或“荧光修复土”特殊地块配置。
  - 实现将残缺耐久武器“种下”并浇注纯水的交互逻辑。
  - 结合时间系统，实现经过一夜后武器耐久恢复满的闭环。

- [ ] **任务 44：流浪商人系统 (Wandering Merchant)**
  - 创建拾荒者老唐 NPC 实体。
  - 编写逻辑使其仅在特定天气（如辐射雷雨天）或特定周期刷新在庇护所边缘。
  - 开发交易面板 UI，支持商品列表随机刷新（稀有种子、水泵升级零件、背包扩容）。

- [ ] **任务 45：新型经济货币系统 (Economy)**
  - 设计并引入“纯净水滴”与“高纯度变异结晶”作为高级货币。
  - 调整怪物掉落、采集收益，将高级货币的产出融入核心循环。

- [ ] **任务 46：植物变异机制 (Plant Mutation)**
  - 监听净化树或庇护所防御状态，当防卫失败时触发变异事件。
  - 将受影响区域的健康植物状态强制切换为“剧毒变异品种”，并替换对应贴图。

- [ ] **任务 47：毒素与基因药剂提炼系统 (Toxin Crafting)**
  - 在工作台增加提炼配方，允许玩家将收集的“剧毒变异品种”作为材料。
  - 实现制造强力毒素武器（高伤害应对重装敌人）与高级基因药剂。

- [ ] **任务 48：性能优化：Data over Nodes 架构设计**
  - 重构植物管理系统，将视线外的植物转为纯字典数据（坐标、生长进度、种类）。
  - 实现当玩家靠近（进入视锥范围）时，动态实例化为 Sprite2D 或简化节点。

- [ ] **任务 49：性能优化：环境特效视锥剔除 (Culling)**
  - 为所有废土环境的 GPUParticles2D 节点添加 VisibleOnScreenNotifier2D。
  - 编写脚本实现离开屏幕时自动停止粒子发射与物理运算。

- [ ] **任务 50：性能优化：全局时间降频运算设计**
  - 修改植物和环境的 Update 逻辑，取消每帧的 _process 调用。
  - 依托全局时间管理器，每经过游戏内设定时间（如 10 分钟），通过信号统一批量更新植物状态。

# 《废土花开》开发任务清单 - 阶段二 (Farming & Combat Loop)

- [ ] **任务 11：基础种田系统 (Tilling & Watering)**
  - 将玩家动作与 TileMap 交互连接，实现挥锄头挖地（转化为“可耕种土地”）。
  - 实现水壶浇水机制，改变土地状态并记录湿润度。

- [ ] **任务 12：植物实体与状态机 (Plant State Machine)**
  - 创建通用的植物实体 (Plant Scene)，包含状态：种子 -> 幼苗 -> 成熟 -> 枯萎/可收获。
  - 接入全局时间系统，根据天数和日夜推进植物生长进度。

- [ ] **任务 13：生存补给类作物实现**
  - 实现“血浆果”、“结晶麦”等基础作物的种植与收获。
  - 添加食用效果：恢复体力、血量，或作为酿造基础材料。

- [ ] **任务 14：防守类战斗植株实现**
  - 实现“荆棘地雷”、“酸液炮仗花”的种植与触发逻辑。
  - 编写陷阱伤害区域与触发冷却机制，用于防御边缘的怪物。

- [ ] **任务 15：共生武备类作物实现**
  - 实现“藤蔓鞭”、“孢子散弹枪”的种植。
  - 实现收获后直接转化为玩家可装备武器的逻辑，并添加耐久度属性。

- [ ] **任务 16：战斗系统基础 (Hitbox & Hurtbox)**
  - 搭建标准化的攻击判定框 (Hitbox) 和受击判定框 (Hurtbox) 组件。
  - 实现玩家的受击判定、扣血、短暂无敌帧和击退效果。

- [ ] **任务 17：武器装备与攻击逻辑**
  - 利用 AnimationTree / AnimationPlayer 实现近战武器（藤蔓鞭）的挥砍动画与多段伤害。
  - 实现远程武器（孢子散弹枪）的弹道发射与消耗逻辑。

- [ ] **任务 18：基础敌人 AI 框架**
  - 创建基础敌人实体 (CharacterBody2D)。
  - 实现基于状态机的 AI 行为：待机巡逻、发现玩家追逐、攻击。

- [ ] **任务 19：夜间辐射潮汐与刷怪机制**
  - 编写全局刷怪管理器。
  - 实现昼夜差异：白天废土怪物迟钝且数量少，夜晚怪物狂暴、索敌范围增加、刷新频率大幅提高。

- [ ] **任务 20：辐射浓度与玩家减益 (Debuff)**
  - 根据玩家所处的环境地块（轻度/重度辐射区），动态增加玩家的辐射积累值。
  - 辐射值过高时，应用减益效果（如体力流失加快、持续扣血、移速降低）。

# 《废土花开》开发任务清单 - 阶段四 (Polish & Release)

- [ ] **任务 31：科技树与进度解锁**
  - 根据成功净化的节点数量，分阶段解锁更高级的植物种子、建筑配方和高级抗辐射能力。
  - 设计进度 UI 界面展示当前废土净化百分比。

- [ ] **任务 32：进阶怪物设计与动作**
  - 设计并实现具有特殊攻击模式的敌人（如会喷吐毒液的多肢畸形人、会钻地的异星节肢动物）。
  - 丰富怪物的受击和死亡动画表现。

- [ ] **任务 33：高压时间与 UI 氛围提示**
  - 在黄昏临近时（如 17:00），在 UI 添加醒目的警告提示（如“夜幕降临，辐射浓度即将上升”）。
  - 配合提示加入心跳声或紧张的音效，催促玩家返回庇护所。

- [ ] **任务 34：物品耐久度与视觉反馈**
  - 完善共生武备（植物武器）的耐久度消耗系统。
  - 在快捷栏 UI 上显示武器残存耐久，并在损坏时播放碎裂音效和粒子。

- [ ] **任务 35：动态环境音效与音乐系统 (Audio System)**
  - 导入并配置背景音乐：庇护所内的治愈音乐，废土白天的荒凉氛围音，夜晚的紧张压迫音乐。
  - 实现根据玩家所在区域和时间平滑过渡音乐的音量混音机制。

- [ ] **任务 36：动作与交互音效 (SFX)**
  - 添加脚步声（草地、废土不同材质）。
  - 添加种田音效（挥锄、浇水、拔出成熟植物）。
  - 添加战斗音效（武器挥舞、命中肉体、怪物嘶吼、玩家受击）。

- [ ] **任务 37：存档与读档系统 (Save/Load)**
  - 实现游戏数据序列化（JSON 或 ConfigFile）。
  - 保存并恢复玩家状态（血量、位置、背包）、TileMap 地形状态、植物生长进度和时间天数。

- [ ] **任务 38：主菜单与过渡界面**
  - 制作精美的标题主菜单（开始游戏、继续游戏、设置、退出）。
  - 添加场景切换时的黑屏淡入淡出 (Fade in/out) 过渡效果。

- [ ] **任务 39：数值平衡与手感调优**
  - 测试并调整作物生长所需的时间与玩家精力消耗的比例。
  - 平衡怪物的血量/伤害与玩家武器输出，确保前期的生存压力和后期的割草爽快感。

- [ ] **任务 40：最终测试与项目导出**
  - 进行完整的游戏流程体验（从开局种田到最终净化核心区域）。
  - 修复边界穿模、状态机卡死等恶性 Bug。
  - 配置项目导出预设，导出 Windows/macOS 执行文件及 Web (HTML5) 体验版。

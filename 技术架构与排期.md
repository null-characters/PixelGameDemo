## 二、 核心数据结构与单例模板 (Core Scripts)

### 1. TimeManager.gd（全局时间管理器，Autoload）
```gdscript
extends Node
class_name TimeManager

signal day_advanced(new_day: int)
signal time_of_day_changed(is_night: bool)

var current_day: int = 1
var time_of_day: float = 0.0  # 0.0=白天开始 → 1.0=夜晚结束
var day_length: float = 600.0  # 10分钟现实=1游戏日（可调）

func _process(delta: float):
    time_of_day += delta / day_length
    if time_of_day >= 1.0:
        time_of_day = 0.0
        current_day += 1
        day_advanced.emit(current_day)
    var is_night = time_of_day > 0.6  # 晚上占40%
    time_of_day_changed.emit(is_night)

func get_time_string() -> String:
    var hour = int(time_of_day * 24)
    return "%02d:00 Day %d" % [hour, current_day]
```

### 2. PlantResource.gd（植物灵魂数据结构，Resource）
```gdscript
@tool
extends Resource
class_name PlantResource

@export var plant_name: String = "血浆果"
@export var growth_days: int = 3
@export var stages: Array[Texture2D] = []  # 种子、幼苗、成熟
@export var harvest_item: String = "blood_berry"
@export var restore_hp: int = 20
@export var restore_stamina: int = 30

# 运行时状态（保存在 FarmingManager 字典中，确保视锥外的数据化存储）
var current_stage: int = 0
var days_grown: int = 0
var is_watered: bool = false
```

### 3. FarmingManager.gd（农田全局管理器，Autoload）
```gdscript
extends Node
class_name FarmingManager

var plants: Dictionary = {}  # key: Vector2i position, value: {resource: PlantResource, node: Node}

func plant_at(pos: Vector2i, plant_res: PlantResource):
    plants[pos] = {"resource": plant_res, "node": null}  # 远处不生成节点，Data over Nodes
    _spawn_plant_if_visible(pos)

func advance_day():
    for pos in plants:
        var p = plants[pos]
        p.resource.days_grown += 1
        if p.resource.days_grown >= p.resource.growth_days:
            p.resource.current_stage = p.resource.stages.size() - 1
        _update_plant_visual(pos)

func _spawn_plant_if_visible(pos: Vector2i):
```

## 五、 The Purge Moment 视觉演出分镜 (10秒 GIF 级呈现)
*目标：打造高情绪价值的正反馈瞬间，便于社群病毒式传播。*

- **0-2s (前奏)**：玩家点击成熟的净化树（近景），树体发出高频金色脉冲。背景音乐为紧张的鼓点。
- **2-4s (爆发)**：冲击波从中心向外剧烈扩散（使用圆形 Shader，半径从 0 瞬间扩大至全屏）。
- **4-7s (翻转)**：TileMap 上的紫黑腐败层像多米诺骨牌般翻转为翠绿草地（使用 `Tween` 颜色渐变结合 `set_cell` 延迟，每格 0.05s 延迟）。
- **7-9s (净化)**：音乐在第 6 秒无缝切换为极为空灵的女声吟唱。空气中的 `GPUParticles2D` 辐射尘瞬间停止发射并淡出。`CanvasModulate` 的全局光照用 0.8s 的时间从压抑的冷蓝色平滑过渡到充满希望的暖橙色。
- **9-10s (余韵)**：镜头轻微拉远，绿色新草地铺满全屏，残存的粒子转化为粉色花瓣随风飘落。
